# 🤖 Discord Голосовой Бот «Алиса»
## Полностью бесплатная версия — пошаговая инструкция

---

## 🆓 Почему это бесплатно?

| Что | Как работает | Цена |
|-----|-------------|------|
| **Discord API** | Официальный бесплатный API | Бесплатно навсегда |
| **edge-tts** (синтез речи) | Использует движок Microsoft Edge | Бесплатно навсегда |
| **YandexGPT Lite** (AI мозг) | При регистрации дают 4000₽ на 60 дней | ~40 млн токенов бесплатно |
| **Встроенные ответы** | Работают без интернета и ключей | Бесплатно всегда |

> 💡 **Итог**: бот будет работать вечно. YandexGPT делает ответы умными — для личного использования 4000₽ гранта хватает на очень долго. Когда кончится — бот переключается на встроенные ответы.

## 📋 Что нам нужно сделать (4 шага)

1. Создать бота в Discord
2. Установить FFmpeg (нужен для звука)
3. Установить Python библиотеки
4. Получить бесплатный ключ Яндекса
5. Настроить `.env` и запустить

## 🔧 Шаг 1 — Создаём бота в Discord

### 1.1 Открой Discord Developer Portal

Перейди по ссылке: **https://discord.com/developers/applications**

Войди в свой Discord аккаунт.

### 1.2 Создай новое приложение

- Нажми кнопку **"New Application"** (синяя, вверху справа)
- Введи имя: например `Алиса`
- Нажми **"Create"**

Ты попадёшь на страницу настроек приложения.

### 1.3 Создай бота

- В левом меню нажми **"Bot"**
- Нажми кнопку **"Add Bot"**
- Нажми **"Yes, do it!"** в подтверждении

### 1.4 Скопируй токен

- На странице Bot нажми кнопку **"Reset Token"**
- Нажми **"Yes, do it!"**
- Появится строка с токеном — нажми **"Copy"**

> ⚠️ **ВАЖНО!** Токен — это пароль твоего бота. Никому не показывай!
> Если кто-то узнает токен, он сможет управлять ботом от твоего имени.

### 1.5 Включи нужные разрешения

На странице Bot, найди раздел **"Privileged Gateway Intents"** и включи три переключателя:

- ✅ **Presence Intent** (включи)
- ✅ **Server Members Intent** (включи)
- ✅ **Message Content Intent** (включи — без этого бот не видит сообщения!)

Нажми **"Save Changes"** (зелёная кнопка).

### 1.6 Добавь бота на свой сервер

- В левом меню нажми **"OAuth2"** → **"URL Generator"**
- В разделе **SCOPES** поставь галочку: ✅ `bot`
- В разделе **BOT PERMISSIONS** поставь галочки:
  - ✅ `Send Messages`
  - ✅ `Read Message History`
  - ✅ `Connect` ← важно для войса!
  - ✅ `Speak` ← важно для войса!
  - ✅ `Use Voice Activity`
  - ✅ `Embed Links` (для красивых сообщений)
- Скопируй ссылку внизу (раздел "Generated URL")
- Открой эту ссылку в браузере
- Выбери свой сервер из списка
- Нажми **"Авторизовать"**

Бот появится на сервере (но пока офлайн).

## 🔊 Шаг 2 — Устанавливаем FFmpeg

**FFmpeg нужен для воспроизведения звука. Без него голос не работает!**

### Windows:

1. Открой сайт: **https://www.gyan.dev/ffmpeg/builds/**
2. Найди раздел **"release builds"**
3. Скачай файл **`ffmpeg-release-essentials.zip`** (около 80MB)
4. Распакуй архив — получишь папку вроде `ffmpeg-7.1-essentials_build`
5. Переименуй папку в просто **`ffmpeg`**
6. Перемести папку в корень диска C: → получится `C:\ffmpeg`
7. Теперь нужно добавить FFmpeg в PATH (чтобы Python нашёл его):
   - Нажми `Win + R`, введи `sysdm.cpl`, нажми Enter
   - Нажми вкладку **"Дополнительно"**
   - Нажми кнопку **"Переменные среды"**
   - В разделе **"Системные переменные"** найди строку **Path**, выдели её
   - Нажми кнопку **"Изменить"**
   - Нажми **"Создать"**
   - Введи: `C:\ffmpeg\bin`
   - Нажми **OK** везде (3 раза)
8. Проверка: открой новый cmd и введи `ffmpeg -version`
   Если видишь версию — всё работает!

### Linux (Ubuntu/Debian/Mint):
```bash
sudo apt update
sudo apt install ffmpeg -y
ffmpeg -version  # проверка
```

### macOS:
```bash
# Сначала установи Homebrew если нет: https://brew.sh
brew install ffmpeg
ffmpeg -version  # проверка
```

## 🐍 Шаг 3 — Устанавливаем Python и библиотеки

### Проверь Python:

Открой командную строку (cmd на Windows, Terminal на Mac/Linux) и введи:
```
python --version
```
Нужна версия **3.10 или новее**. Если нет — скачай с **https://python.org**

> На Windows при установке Python обязательно поставь галочку **"Add Python to PATH"**!

### Скачай файлы бота:

Положи все файлы бота в одну папку, например `C:\alisa-bot\` (Windows) или `~/alisa-bot/` (Linux/Mac).

### Установи библиотеки:

Открой командную строку **в папке с ботом** и выполни:

```bash
pip install -r requirements.txt
```

Подожди пока всё установится (1-2 минуты). Ты должен увидеть много текста и в конце "Successfully installed...".

> Если ошибка "pip не найден" — попробуй `python -m pip install -r requirements.txt`

## 🔑 Шаг 4 — Получаем бесплатный ключ Яндекса

Это самый важный шаг для умных ответов. Разберём подробно.

### Зачем нужен ключ?

Без ключа бот работает, но отвечает только шаблонными фразами (привет, пока, который час...).
С ключом — отвечает на любые вопросы как Алиса, помнит контекст разговора.

### Что дают бесплатно?

При первой регистрации в Yandex Cloud **автоматически дают 4000₽ на 60 дней**.
YandexGPT Lite стоит **0.10₽ за 1000 токенов** (~700-800 слов).
Считай: `4000 / 0.10 × 1000 = 40 000 000 токенов` — для личного бота это очень много!

### 4.1 Создай аккаунт в Yandex Cloud

1. Перейди на **https://cloud.yandex.ru**
2. Нажми **"Войти"** — войди через Яндекс аккаунт (тот же что и для Яндекс Почты)
3. Если нет Яндекс аккаунта — создай на **https://passport.yandex.ru**

### 4.2 Создай платёжный аккаунт (грант придёт сюда)

> Нужна карта только для верификации. Деньги снимут только если ты сам перейдёшь на платный тариф и у тебя закончится грант.

1. В левом меню найди **"Биллинг"** или перейди: **https://console.yandex.cloud/billing**
2. Нажми **"Создать платёжный аккаунт"**
3. Заполни форму:
   - Тип: **Физическое лицо**
   - Привяжи карту (Visa, Mastercard, МИР)
4. После создания на счёт автоматически придут **4000₽ в виде гранта**

### 4.3 Создай каталог (Folder)

Каталог — это контейнер для всех ресурсов. Folder ID нам понадобится.

1. В консоли нажми **"Облако"** в левом меню
2. Нажми **"Создать каталог"**
3. Введи имя: например `discord-bot`
4. Нажми **"Создать"**
5. **Скопируй ID каталога** — это строка вида `b1gxxxxxxxxxxxx`
   (видна в строке браузера или на странице каталога)

### 4.4 Создай сервисный аккаунт

Сервисный аккаунт — это "пользователь для программ", от его имени бот будет делать запросы.

1. В левом меню выбери **"Сервисные аккаунты"** (или через поиск)
2. Нажми **"Создать сервисный аккаунт"**
3. Имя: например `discord-alisa`
4. В поле **"Роли"** нажми **"Добавить роль"** и найди: `ai.languageModels.user`
   (это разрешение на использование YandexGPT)
5. Нажми **"Создать"**

### 4.5 Создай API ключ

1. Нажми на созданный сервисный аккаунт
2. Перейди на вкладку **"API-ключи"**
3. Нажми **"Создать API-ключ"**
4. Описание: например `для discord бота`
5. Нажми **"Создать"**
6. Появится окно с ключом — **сразу скопируй его!** (он показывается только один раз)
   Ключ выглядит так: `AQVN...длинная строка...`

### Создай файл .env

В папке с ботом создай файл с именем `.env` (именно с точкой в начале!).

**Windows:** Открой Блокнот → Сохранить как → выбери "Все файлы" → введи имя `.env`


## 💬 Шаг 7 — Пользуемся ботом

### Текстовый режим (работает сразу):

Просто напиши в любом канале:
```
Алиса, как дела?
Алиса, сколько времени?
Алиса, расскажи анекдот
```

Или используй команды:
```
!ask Что такое квантовая физика?
!помощь
!статус
```

### Голосовой режим (нужен FFmpeg):

1. Зайди в голосовой канал
2. Напиши в текстовый чат: `!join`
3. Бот зайдёт к тебе и поздоровается голосом
4. Напиши: `!listen` (бот начнёт слушать)
5. Теперь пиши `Алиса, вопрос` — бот ответит голосом!

### Все команды:

| Команда | Что делает |
|---------|-----------|
| `!join` или `!войди` | Зайти в голосовой канал |
| `!leave` или `!уйди` | Выйти из канала |
| `!listen` или `!слушай` | Включить прослушивание |
| `!stop` или `!стоп` | Остановить |
| `!ask [вопрос]` | Спросить текстом |
| `!reset` или `!сброс` | Очистить историю диалога |
| `!статус` | Показать статус (работает ли AI) |
| `!помощь` | Список команд |

## ❓ Частые проблемы и решения

### "DISCORD_TOKEN не найден"
→ Проверь что файл называется `.env` (не `.env.txt`!)
→ Проверь что токен без лишних пробелов

### Бот не отвечает в Discord
→ Убедись что включён **Message Content Intent** в Developer Portal
→ Перезапусти бота

### Бот не говорит голосом
→ Убедись что FFmpeg установлен: введи `ffmpeg -version` в cmd
→ Убедись что установлен PyNaCl: `pip install PyNaCl`
→ Убедись что у бота есть права **Connect** и **Speak** в канале

### "Unauthorized" или "401" от Яндекса
→ Проверь API ключ — скопирован ли он полностью
→ Создай новый API ключ (шаг 4.5)

### Бот отвечает шаблонными фразами, не AI
→ Запусти `!статус` — там написано работает ли YandexGPT
→ Проверь правильность YANDEX_API_KEY и YANDEX_FOLDER_ID в .env
→ Это нормально если грант закончился — бот всё равно работает

### Ошибка при установке PyNaCl
→ Windows: `pip install PyNaCl --pre`
→ Или скачай готовый whl: https://www.lfd.uci.edu/~gohlke/pythonlibs/
